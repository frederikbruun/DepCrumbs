Installation

Two ways to get started — pick whichever fits your workflow.

Plugin

Add DepTrace as a Claude Code plugin for automatic tracking.

claude plugin add deptrace

Installs hooks automatically. No further setup needed.

CLI

Install the CLI for manual control and scripting.

npm install -g deptrace
deptrace init
deptrace setup

Run deptrace status to verify installation.
